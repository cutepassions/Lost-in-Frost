## 🎮 서비스 소개

- 평범한 생존게임은 가라! 다양한 상호작용과 랜덤 카드 선택을 통한 효과를 챙겨 30분을 버텨라!

## 팀원 소개

- 김범창 (FE)
- 진병욱 (BE, CI/CD)

## 📆 개발 기간

- 2023.10.09 ~ 11.17 (6주)

## 💻 사용 서버

### Backend

- Spring Cloud
